Database steps:
1. Create a new database
2. Replace the database name 'digilibSensusLibrary' from the file '/database/digilibSensusLibrary.sql' with the one of the newly created database
3. Run the script from the /database/digilibSensusLibrary.sql 

Bin setup: The bin folder should have the necessary dlls to run the project. If the bin folder is empty, extract the bin.rar archive contents in the bin folder.
